1. Как стартует приложение .Net Framework

1.1. Windows анализир PE-заголовок exe-шника.
Это стандартный Windows заголовок и в нем описано, что это .Net приложение и его версия.
1.2. Загружается в память основная библиотека MSCorEE.dll.
Она же загружает всю инфраструкту JIT-компилятор, GC и т.п.Аш
1.3. С помощью метаданных подгружаются все зависимости.
1.4. Запускается код.
